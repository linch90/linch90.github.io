¶Automatic URL Linking

多数Markdown处理器会自动将URL转换成链接。例如，你输入了www.example.com，那么Markdown处理器会自动将其转换为 www.example.com

¶Disabling Automatic URL Linking

如果你不想将URL形式的字符串转换为链接，只需要在URL两侧标注 `字符即可

www.example.com
